This plugin allows you to easily view and edit custom fields in bulk for any post type directly from the WordPress admin.

== Description ==

Bulk Custom Fields Editor helps administrators quickly manage custom fields (post meta) for multiple posts at once.

You can select the post type, define which meta keys to edit, and save changes for all posts shown on the page.

== Installation ==

Upload the "Bulk Bulk Custom Fields Editor" plugin to your blog, activate it from the Plugins menu, then access the "Bulk Custom Fields Editor" menu in your WordPress admin dashboard.
